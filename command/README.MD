### 工厂方法模式（Factory Method pattern）

##### 创建模式（Creational Pattern）


#### 定义

***Exposes a method for creating objects, allowing subclasses to control the actual creation process.***

***定义一个用于创建对象的接口，让子类决定实例化哪一个类。工厂方法模式使一个类的实例化延迟到其子类。***
