### 策略模式（Strategy Pattern）

##### 行为模式（Behavioural Pattern）

#### 定义

***Defines a set of encapsulated algorithms that can be swapped to carry out a specific behavior.***

***定义了一组算法，将每个算法都封装起来，并且使它们之间可以互换。***
